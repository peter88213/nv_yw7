# nv_yw7

The [novelibre](https://github.com/peter88213/novelibre/) Python program helps authors organize novels.  

*nv_yw7* is a plugin providing *.yw7* file import and export. 

![Screenshot](docs/Screenshots/screen01.png)

## Requirements

- [novelibre](https://github.com/peter88213/novelibre/) version 2.0+

## Download and install

[Download the latest release (version 2.0.0)](https://github.com/peter88213/nv_yw7/raw/main/dist/nv_yw7_v2.0.0.zip)

- Extract the "nv_yw7_v2.0.0" folder from the downloaded zipfile "nv_yw7_v2.0.0.zip".
- Move into this new folder and launch **setup.pyw**. This installs the plugin for the local user.
